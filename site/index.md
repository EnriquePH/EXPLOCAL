## Descargar el código fuente

El código fuente original de 1996 (Borland C++ 3.1 + Object Windows 1.0),
los datos, los ejemplos y el ejecutable para Windows 3.1 están disponibles
en el repositorio de GitHub:

- [**Explorar y descargar `legacy/`**](https://github.com/EnriquePH/EXPLOCAL/tree/master/legacy) — código fuente (`EXPLOCAL.CPP`, `CALCULOS.H`...), datos (`*.DAT`), recursos (`*.RC`, `*.DLG`) y el compilado `EXPLOCAL.EXE`.
- [Descargar `EXPLOCAL.EXE` directamente](https://github.com/EnriquePH/EXPLOCAL/raw/master/legacy/EXPLOCAL.EXE)
- [Descargar el repositorio completo (.zip)](https://github.com/EnriquePH/EXPLOCAL/archive/refs/heads/master.zip) — incluye también los documentos originales de la memoria (`docs/`) y los ficheros fuente del sistema de ayuda (`docs/help/`).
